Installing Kimchi

First of all, download Kimchi package according to your operating system.

We provide packages for the latest version of Fedora, openSUSE, Ubuntu and RHEL. Those packages were built on and for x86_64 machines.

After it, install Kimchi package using the default packaging management tool in your system. You can do it by command line like below:

Fedora and RHEL:

sudo yum install <kimchi.rpm>

Ubuntu:

sudo dpkg -i <kimchi.deb>

openSUSE:

sudo zypper install <kimchi.rpm>

Starting kimchid service

After installing kimchi package, you just need to start its service to get it up:

sudo service kimchid start

Note: The above command line may vary depends on your operating system.

Then access Kimchi in your favorite browser https://<machine-ip>/8001

Have fun! =)
